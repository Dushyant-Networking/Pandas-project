# Car Sales Analysis

## Objective
Analyze car sales data to identify top-performing brands, models, and trends.

## Tools Used
- Python
- Pandas

## Key Insights
- Top 10 car models by sales
- Brand-wise revenue contribution
- Price vs sales relationship

## Project Structure
- data/: car_prices.csv
- notebooks/: car data.py
- output/: Brandwise report.csv

## How to Run
1. Install requirements  
   pip install -r requirements.txt
2. Open notebook and run all cells
